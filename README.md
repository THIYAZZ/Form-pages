# Login & Sign Up Form Page

This project is a modern login and sign up form page created using HTML and CSS.
It features a toggle-based interface that allows users to switch between
login and registration forms.

## Features
- Login and Sign Up toggle functionality
- Clean and modern dark-themed UI
- Styled input fields with icons
- Responsive and user-friendly layout

## Technologies Used
- HTML5
- CSS3

## Purpose
This project was developed to practice front-end form design,
UI styling, and layout structuring using HTML and CSS.
