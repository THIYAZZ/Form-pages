# Document Upload Form

This project is a simple document submission form built using HTML and CSS.
It allows users to upload a document or provide a form link and proceed.

## Features
- File upload option
- Paste form link option
- Clean and minimal UI design
- User-friendly layout with clear actions

## Technologies Used
- HTML5
- CSS3

## Purpose
This project was developed to practice form elements, file input handling,
and front-end UI design using HTML and CSS.
